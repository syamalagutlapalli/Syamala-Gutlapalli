# Human Development Index (HDI) Prediction Using Machine Learning

## 📌 Project Overview

The Human Development Index (HDI) Prediction System is a Machine Learning-based web application that predicts the Human Development Index (HDI) of a country using important socio-economic indicators such as Life Expectancy, Mean Years of Schooling, Gross National Income (GNI) per Capita, and Internet Users.

The project uses a Linear Regression model trained on historical HDI data and is deployed using the Flask web framework to provide real-time predictions through a user-friendly interface.

---

## 🎯 Objectives

- Predict the Human Development Index (HDI) of a country.
- Analyze the impact of socio-economic indicators on HDI.
- Build a machine learning prediction model.
- Deploy the trained model using Flask.
- Provide an interactive web interface for users.
